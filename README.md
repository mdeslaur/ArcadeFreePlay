# Patches for arcade games to add Free Play

These directories contain Free Play patches for arcade game roms.

They currently support:
- 1943
- Arkanoid
- Atari Starship 1
- Bank Panic
- Bombjack
- Bubble Bobble (Redux)
- Bubble Bobble Lost Cave
- Double Dragon
